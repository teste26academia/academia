# Diagnóstico do login Firebase

Substitua apenas o `app.js` do seu projeto por este arquivo para ver o erro real do login por telefone.

O que este arquivo faz:
- Mostra o código real do Firebase no alerta
- Mostra o diagnóstico na tela
- Cria o reCAPTCHA só quando o login é aberto
- Mantém painel admin, aluno e financeiro

Se o problema continuar, o texto exibido na caixa de diagnóstico vai indicar exatamente a causa.
