# Cadastro público de alunos

Este pacote permite que os alunos criem a própria conta:

- login por telefone
- cadastro com nome/faixa/foto
- perfil salvo no Firebase
- aprovação do admin
- área do aluno
- lista de pendentes/aprovados

## Arquivos
- index.html
- app.js
- manifest.json
- service-worker.js

## Importante
No Firebase Authentication:
- ative Phone

No Firebase Auth Authorized domains:
- adicione o domínio do GitHub Pages

## Próximo passo
Se quiser, a próxima versão pode incluir:
- upload real da foto no Firebase Storage
- aprovação por link/QR Code
- regras de segurança por usuário
