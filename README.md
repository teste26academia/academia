# Liga Garra de Águia PG — Fase 1 e 2

Esta versão já une as duas primeiras fases:
- Login por telefone no Firebase
- Painel administrativo protegido por senha
- Firestore com cadastro online de alunos
- Presença sincronizada
- Área do aluno
- Exportar/importar dados
- Visual responsivo

## Importante
No Firebase Authentication:
- ative o método **Telefone**
- adicione o domínio do GitHub Pages em **Authorized domains**
